## 🛠️ Build & Development

### Requirements
- **Android Studio** - Latest version recommended
- **Java 8+** - Java 8 or higher
- **Android SDK** - API level 21 (Android 5.0) minimum
- **Gradle** - Gradle 8.0 or higher

### Building from Source
```bash
git clone <repository-url>
cd TestProject
./gradlew build
./gradlew installDebug
```

### APK Output
- **Debug APK** - `/apk/debug_1.0.apk`
- **Release APK** - `/apk/release_1.0.apk`

## 🎯 Usage Guide

### Getting Started
1. **Installation** - Install the APK from releases or build from source
2. **Navigation** - Use the URL bar for addresses or searches
3. **Tabs** - Tap the "+" button to create new tabs
4. **Menu** - Access all features through the menu button

### Feature Usage
- **Dark Mode** - Menu → Dark Mode
- **Private Browsing** - Menu → Private Tab
- **Reading Mode** - Menu → Reading Mode
- **User Agent** - Menu → User Agent
- **Performance** - Menu → Performance
